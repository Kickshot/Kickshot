# Kickshot

## Controls

| Key         | Action         |
|-------------|----------------|
| WASD        | Move           |
| LCtrl       | Crouch         |
| Space       | Jump           |
| R           | Reload         |
| Left Click  | Primary Fire   |
| Right Click | Secondary Fire |
| Mouse Wheel | Change Weapons |
| Escape      | Exit game      |
